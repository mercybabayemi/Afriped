MEMORY UNIT
Memory is a device that has the capacity to store information. A memory unit is the amount of data that the memory can hold. Besides, we measure this storage capacity in terms of bytes. The computer memory is the storage location where data programs and information are stored.
There are two types of memory.
PRIMARY/MAIN MEMORY: The main memory is the internal storage area of the computer system. It is where information and programs are stored for immediate processing by the CPU.
Two types of primary storage:
ROM- It is a non-volatile memory that retains data stored on it even after the computer is powered off. The data inside the ROM cannot be changed. The operating system boot records are stored inside the ROM. There are three (3) different variations, this include:
PROM that is Programmable ROM.
EPROM that is Erasable Programmable ROM.
EEPROM that is Electrically Erasable Programmable ROM.
RAM- It is the part of the primary memory that holds the data and programs that the CPU is processing. The CPU reads and writes into the RAM as often as required. They are volatile because the content is lost when the computer is powered off.
SECONDARY MEMORY: This is also called the auxiliary or external memory. This is any storage device that is used to store data, information and programs outside the primary memory. It is not directly accessible by the Central Processing Unit. The computer usually uses its input/output channels to access secondary storage. They are non-volatile. Examples of secondary storage:
HDD (Hard Disk Drive): Magnetic storage for long-term data retention.
SSD (Solid State Drive): Faster, non-volatile storage using flash memory.
Optical Discs (CDs, DVDs): used for data storage.
USB Flash Drives: Portable storage devices using flash memory.
