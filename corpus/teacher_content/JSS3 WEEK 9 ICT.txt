GRAPH
A graph is a visual representation of data. It helps in illustrating relationships, trends, and comparisons between sets of data by converting numerical data into visual formats, making it easier to understand and analyze.
Types of Graphs (Charts) in a Spreadsheet Package
Bar Chart
Pie Chart
Line Graph
Histogram
How to Create a Graph in Excel
Step 1: First, input the data into Excel.
Step 2: Go to the 'Insert' tab and choose a graph type.
Step 3: Select the data you want to use.
Step 4: Click 'Insert' to add your chosen graph.
BAR CHART:
A bar chart represents data with rectangular bars. Each bar's length is proportional to the value it represents, and the bars can be plotted either horizontally or vertically.
Example: Showing the number of students in different classes.
Steps to create a bar chart
Enter the data in Excel.
Select the data range.
Go to Insert → Bar Chart → Choose a chart type (e.g., Clustered Bar).
Customize the chart title, axes, and legend.
Change chart style and design if desired.

LINE CHART:
A line chart displays data points connected by straight lines. It is useful for showing trends or changes over time.
Example: Showing the temperature changes over a week or tracking a student’s grades over a semester.
Steps to create a line graph:
Organize your data into a table with rows for the X and Y variables.
Choose your graphing tool
Select the data and choose the "Line Chart" option to create the graph.
Label the axes and title the chart appropriately.
Editing Line Graphs:
Change data range: Adjust the data series if you need to add or remove data points.
Modify axes: Format the X and Y axes by adjusting scales, changing labels, or modifying the axis title for clarity.
Add markers: Highlight specific data points with markers for better visualization.
Add gridlines: Include or modify gridlines for easier interpretation of the data.
