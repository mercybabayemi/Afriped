ASSIGNMENT
What are input devices
Types of Input Devices:
1. Keyboard:
Description: A keyboard is a peripheral device consisting of keys that allow the user to input characters, symbols, and perform functions.
Types:
Mechanical Keyboard: Uses physical switches to register keystrokes.
Membrane Keyboard: Uses pressure pads to sense keystrokes.
Common Uses: Typing text, entering commands, gaming.
2. Mouse:
Description: A mouse is a pointing device used to interact with items on a screen, usually represented by a cursor.
Types:
Optical Mouse: Uses light sensors to detect movement.
Laser Mouse: Uses a laser for more precise movement tracking.
Common Uses: Selecting items, dragging files, navigating through interfaces.
3. Touchpad:
Description: A touch-sensitive surface that detects finger movements and translates them into cursor movement.
Common Uses: Built into laptops as a mouse alternative for navigation and gestures.
4. Touchscreen:
Description: A display that also serves as an input device, allowing users to interact directly by touching the screen.
Types:
Resistive: Uses pressure to detect input.
Capacitive: Uses the electrical conductivity of the human body to detect input.
Common Uses: Smartphones, tablets, kiosks.
5. Scanner:
Description: A device that captures images or text from physical documents and converts them into a digital format
Types:
Flatbed Scanner: Scans documents one at a time.
Sheet-fed Scanner: Feeds documents automatically for scanning.
Common Uses: Digitizing documents, copying photographs.

Description: A device that captures sound and converts it into electrical signals for input into a computer.

7. Webcam:
Description: A video input device used to capture live video footage, typically for video calls or streaming.
Common Uses: Video conferencing, live streaming, security monitoring.
8. Game Controller:
Description: An input device specifically designed for interacting with video games.
Types:
Gamepad: A handheld controller with buttons and joysticks.
Joystick: A stick-like controller for movement in games.
Common Uses: Gaming, simulations, virtual reality.
9. Stylus:
Description: A pen-like input device used for writing or drawing on touchscreen devices.
Common Uses: Drawing tablets, smartphones, tablets for digital art, note-taking.
10. Barcode Reader:
Description: A device that scans barcodes and inputs the data into a system.
Common Uses: Retail checkout systems, inventory management.
