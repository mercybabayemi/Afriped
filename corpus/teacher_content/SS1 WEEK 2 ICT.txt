DIFFERENCES BETWEEN RAM AND ROM
Random Access Memory (RAM)
Read-Only Memory (ROM)
Memory
Volatile Memory: Loses contents when powered off.
Non-Volatile Memory: Retains contents when powered off.
Storage
Temporary Storage: Stores data and instructions for active tasks
Permanent Storage: Stores unmodifiable data, typically firmware.
Speed
Very fast, allowing quick read/write access.
Slower than RAM.
Types
Includes Static RAM (SRAM) and Dynamic RAM (DRAM), with DRAM being more common.
Includes PROM, EPROM, and EEPROM.
Function
Stores data for running applications and the operating system.
Contains BIOS for booting and hardware initialization.
SOFTWARE
The software is a term used to describe the collection (or set) of programs that can be run on a computer system. A program is a set of instructions that tells the computer what to do, when to do it and how to do it. There are two types of software, these include:
System software: This software controls the computer hardware and the internal operations of the computer system. Examples include Windows OS, Linux OS, MAC OS etc
Application software: These are software that enables the user to perform specific operation or task on the computer. Examples include: MS Word, MS Excel,  Corel draw, MS Access, MS PowerPoint, Impress, Prezi, etc,
Utility Software: Programs that perform maintenance tasks (e.g., antivirus software, disk cleanup tools).
HUMANWARE/PEOPLEWARE
Humanware or Peopleware refers to the human aspect of computer systems, encompassing the individuals who design, develop, manage, and use the hardware and software components. This includes: System Analysts, Programmers/Developers, Network Administrators, Database Administrators, IT Support Technicians, etc.

CHARACTERISTICS OF A COMPUTER
Speed: Computers can process data and execute instructions at extremely high speeds, often measured in gigahertz (GHz).
Accuracy: Computers perform calculations and operations with a high degree of accuracy, reducing human error.
Versatility: Computers can perform a wide range of tasks, from simple calculations to complex simulations.
Reliability/Durability: Computers do not get tired like human; they can work continuously for days and weeks without any error.
Memory and storage: Computers have vast storage capacities, both temporary (RAM) and permanent (HDD).
Programmable: All tasks and activities performed by a computer are made possible by means of a program.
Automation: Once programmed, computers can perform tasks automatically without human intervention.
Diligence: Unlike humans, computers do not suffer from fatigue and can perform repetitive tasks consistently without error.
Connectivity: Computers can connect to other systems and networks, enabling communication and data exchange.
Multitasking: Modern computers can run multiple applications simultaneously, making them efficient for various tasks.
