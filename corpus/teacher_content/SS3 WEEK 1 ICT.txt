NETWORKING
Networking is the practice of connecting computers and devices to share resources like data, applications, and internet access. This connectivity enables communication and collaboration over local or global distances. A computer network, or simply a network, is a group of interconnected devices linked by communication channels to facilitate resource sharing.
USES OF NETWORKING
File Sharing: Sharing files and data between computers.
Resource Sharing: Sharing resources like printers and internet connections.
Communication: Enabling communication via email, instant messaging, and video conferencing.
Data Backup: Centralized data storage for backup and recovery.
Remote Access: Accessing a computer or network from a remote location.
TYPES OF NETWORKING
Local Area Network (LAN): Connects devices within a small geographic area like a building or campus.
Wide Area Network (WAN): Covers broad areas, such as cities or countries, often connecting multiple LANs.
Metropolitan Area Network (MAN): Spans a city or large campus, linking multiple LANs within a metropolitan area.
Personal Area Network (PAN): Connects personal devices over short distances, such as Bluetooth.
Campus Area Network (CAN): Connects multiple buildings within a campus, extending LANs across a larger area.
Home Area Network (HAN): Connects devices within a home, such as computers and smart devices.
Global Area Network (GAN): Links multiple WANs and LANs globally for worldwide connectivity.
NETWORK DEVICES
Network devices, also known as nodes, are essential components in a network that facilitate connection and communication. The most common network devices include routers, switches, hubs, modems, and network interface cards. Each of these devices plays a specific role in ensuring the smooth operation of a network.
Router: Directs data between different networks.
Switch: Connects devices within a network and directs data to specific destinations.
Hub: Broadcasts data to all devices in a network segment.
Modem: Converts digital data to analog signals for internet access.
NIC (Network Interface Card): Enables a computer to connect to a network.
Bridge: Connects similar networks to function as one.
Access Point: Provides wireless connectivity to a wired network.
Firewall: Protects the network by controlling traffic based on security rules.
ADVANTAGES OF NETWORKING
Resource Sharing: Reduces costs by sharing resources.
Improved Communication: Facilitates fast communication between users.
Centralized Management: Simplifies data and resource management.
Enhanced Connectivity: Promotes global collaboration.
Data Sharing: Enables simultaneous access to shared data and applications.
DISADVANTAGES OF NETWORKING
Security Risks: Vulnerable to hacking and data breaches.
Complexity: Requires technical expertise to set up and maintain.
Cost: Can be expensive, especially for large networks.
Reliability: Dependent on the performance of network devices, with potential for disruptions.

NETWORK TOPOLOGY
In computer networking, topology refers to the arrangement of connected devices within a network. Network topologies are categorized into several basic types:
Bus Topology: In a bus topology, all devices are connected to a single central cable, known as the backbone. This backbone acts as a shared communication medium. Example: Ethernet.
Advantages
Easy to connect additional devices or peripherals.
Requires less cable than star topology.
Disadvantages
The entire network shuts down if there is a break in the main cable.
Troubleshooting network failures can be challenging.
Star Topology: In a star topology, all devices are connected to a central hub, switch, or router, forming a star-like structure. Example ARCNET.
Advantages
Easy to install and wire.
Faults are easy to detect and isolate.
Disadvantages
Requires more cable length than a bus topology.
More expensive due to the cost of hubs and additional cabling.
Ring Topology: In a ring topology, each device is connected to two other devices, forming a circular or ring-like structure. Example: IBM Token Ring.
Advantage
Simple data flow with predictable performance.
Disadvantage
A single failure can impact the entire network.
Mesh Topology: Devices are interconnected with multiple paths for redundancy and reliability.
Advantages
Multiple paths for data ensure high reliability; failure of one device or connection doesn’t affect the network.
Adding new devices is straightforward without disrupting existing connections.
Disadvantages
High cost due to extensive cabling and hardware.
Troubleshooting is challenging because of the multiple data pathways.
Tree Topology: A hierarchical structure with a central root node, from which branches of connected nodes extend, resembling a tree. This topology combines characteristics of both star and bus topologies.
Advantages
Adding new devices is simple and doesn't impact network performance.
Centralized management allows for easier troubleshooting.
Disadvantages
Failure of the root or main branch node can disrupt communication for entire segments.
More complex to configure and maintain compared to simpler topologies.
Hybrid Topology: A hybrid topology in networking refers to a network structure that combines two or more different types of topologies, such as star, ring, bus, or mesh, to form a complex and more adaptable network design.
Advantages
Leverages the strengths of multiple topologies for a customized and resilient network.
Provides redundancy and fault tolerance, keeping the network operational even if one part fails.
Disadvantages
Increases complexity, requiring advanced knowledge for setup and maintenance.
Troubleshooting can be challenging due to the diverse and complex topology mix.
