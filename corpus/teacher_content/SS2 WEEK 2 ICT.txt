COMPONENTS OF THE CPU
The CPU comprises three main components, each working together to execute its functions:
Arithmetic Logic Unit: This is the part of the CPU that performs all the arithmetic and logical computations.
Control Unit: This unit is responsible for executing the results coming out of the ALU. It directs all the operations of the various components that make up the computer system, to ensure that they work together in harmony.
Memory Unit: This is also called register. It is the storage area for instructions or data within the processor.
Functions of the Arithmetic Logic Unit (ALU):
- Executes arithmetic operations (addition, subtraction, multiplication, division).
- Performs logical operations (AND, OR, NOT, XOR) and comparisons.
Functions of the Control Unit (CU):
- Fetches, decodes, and directs instructions.
- Manages data flow between the CPU, memory, and input/output devices.
- Oversees the operation of input and output (I/O) devices.
- Interprets instructions.
Functions of the Memory Unit (Registers):
- Stores data and instructions temporarily or permanently.
- Facilitates quick data access for the CPU.

Main Memory includes:
Random Access Memory (RAM): Volatile, temporary storage for active tasks.
Read-Only Memory (ROM): Non-volatile, permanent storage for essential firmware.
DIFFERENCES BETWEEN RAM AND ROM
Random Access Memory (RAM)
Read-Only Memory (ROM)
Memory
Volatile Memory: Loses contents when powered off.
Non-Volatile Memory: Retains contents when powered off.
Storage
Temporary Storage: Stores data and instructions for active tasks
Permanent Storage: Stores unmodifiable data, typically firmware.
Speed
Very fast, allowing quick read/write access.
Slower than RAM.
Types
Includes Static RAM (SRAM) and Dynamic RAM (DRAM), with DRAM being more common.
Includes PROM, EPROM, and EEPROM.
Function
Stores data for running applications and the operating system.
Contains BIOS for booting and hardware initialization.
