HARDWARE COMPONENTS
Hardware components are the physical, tangible parts of a computer system. They are the devices and equipment that make up the computer's structure.
Types:
1.  Input Devices: These devices allow users to input data into the computer. Examples include:
Keyboard
Mouse
Scanner
Microphone
Webcam
2. Output Devices: These devices display or present the processed data to the user. Examples include:
Monitor
Printer
Speaker
Projector
3. Processing Unit (CPU): This is the "brain" of the computer, responsible for executing instructions and performing calculations.
Memory: This stores data and instructions for the CPU to access. There are two main types:
RAM (Random Access Memory): Temporary storage for data and instructions.
ROM (Read-Only Memory): Permanent storage for essential system data.
