OVERVIEW OF COMPUTER SYSTEM
Definition of a Computer
A computer can be defined as an electronic device which under the control of a stored program accepts data, stores the data, processes the data and brings out the results as meaningful information.
COMPUTER AS AN IPOS SYSTEM
Computer performs 4 key operations.
a. Input b. Processing c. Output d. Storage
Input: Data are entered into the computer through the input unit. Data are raw facts that are to be processed into meaningful form.
Processing: Computer performs certain operations on data entered into it; the Central Processing Unit (CPU) handles this operation.
Output: The result of what has been processed will be displayed by the output device such as the monitor.
Storage: Inputted data and processed information are stored in the memory and permanent storage on the computer.
CONSTITUENTS/COMPONENTS OF A COMPUTER SYSTEM
There are 3 main constituents of the computer system, these are:
The Hardware
The Software
The Humanware/Peopleware

HARDWARE
Computer hardware consists of the physical components required to perform tasks and execute software.
HARDWARE COMPONENTS
The computer hardware components can be classified as:
Input Devices: Input devices provide data and control signals to a computer. Examples include: Keyboard, Mouse, Scanner, Microphone, Webcam, Joystick, Touchscreen
Output Devices: Output devices translate data from the computer into a form humans can understand. Examples include: Monitor, Printer, Speaker, Projector, Headphones
Storage Devices: Storage devices store data permanently or temporarily. Examples include: Hard Disk Drive (HDD), CDs, DVDs, USB, Flash Drive, Memory Card
Processing Devices: Processing devices interpret and execute instructions. The CPU, considered the brain of the computer, includes: CPU, Motherboard, Sound Card, Network Card
CPU Components: - The CPU is divided into three main parts:
Control Unit (CU)
- Fetches, decodes, and directs instructions.
- Manages data flow between the CPU, memory, and input/output devices.
Arithmetic and Logic Unit (ALU)
- Executes arithmetic operations (addition, subtraction, multiplication, division).
- Performs logical operations (AND, OR, NOT, XOR) and comparisons.
Memory Unit
- Stores data and instructions temporarily or permanently.
- Facilitates quick data access for the CPU.
Main memory includes:
Random Access Memory (RAM): Volatile, temporary storage for active tasks.
Read-Only Memory (ROM): Non-volatile, permanent storage for essential firmware.
