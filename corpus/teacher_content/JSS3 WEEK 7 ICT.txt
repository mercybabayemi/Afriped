ADDING FORMULAS AND PERFORMING CALCULATIONS IN EXCEL
To indicate that you are entering a formula in Excel, you must start with a specific symbol. In Microsoft Excel, formulas always begin with the "=" sign.
USING FUNCTIONS IN MICROSOFT EXCEL
Excel offers various built-in functions to simplify calculations:
SUM Function
The SUM function adds up the values in a range of cells.
Syntax: =SUM(First cell:Last cell)
AVERAGE Function
The AVERAGE function calculates the mean of values in a range of cells.
Syntax: =AVERAGE(First cell:Last cell)
COUNT Function
The COUNT function counts the number of numeric entries in a specified range.
Syntax: =COUNT(First cell:Last cell)
MAX Function
The MAX function returns the largest value from a set of values within a range of cells.
Syntax: =MAX(First cell:Last cell)
MIN Function
The MIN function returns the smallest value from a set of values in a given range.
Syntax: =MIN(First cell:Last cell)

HOW TO SUM NUMBERS
Open Excel and enter your numbers into cells. For example, enter numbers into cells A1, A2, A3, and A4.
Select the Cell for the Sum: Click on the cell where you want the sum to appear. For instance, cell A5.
Enter the SUM Function: In the selected cell, type =SUM(A1:A4)
HOW TO FIND PRODUCT ON MS EXCEL
Open Excel and enter your numbers into cells. For example, enter numbers into cells A1, A2, A3, and A4.
Select the Cell for the Product: Click on the cell where you want the product to appear. For instance, cell A5.
Enter the PRODUCT Function: In the selected cell, type =PRODUCT(A1:A4)
HOW TO SUBTRACT ON MS EXCEL
Open Excel and enter your numbers into cells. For example, enter a number into cell A1 and another number into cell A2.
Select the Cell for the Result: Click on the cell where you want the result to appear. For instance, cell A3.
Enter the Subtraction Formula: In the selected cell, type =A1 - A2.
HOW TO DIVIDE ON MS EXCEL
Open Excel and enter your numbers into cells. For example, enter a number into cell A1 and another number into cell A2.
Select the Cell for the Result: Click on the cell where you want the result to appear. For instance, cell A3.
Enter the Subtraction Formula: In the selected cell, type = A1/A2
