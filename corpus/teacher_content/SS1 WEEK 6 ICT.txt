INPUT DEVICES
Computer input devices are hardware components that allow users to enter data and instructions into a computer. They are essential for interacting with the computer and performing various tasks.
Examples:
- Keyboard
- Mouse
- Touchscreen
- Scanner
- Webcam
- Microphone
- Joystick
- Gamepad
- Trackball
- Stylus and Lighten

Features of Input Devices
- Input type: Keyboards and mice use physical buttons and movement, touchscreens use touch, scanners use light, and microphones use sound.
- Interface: Input devices can be wired or wireless.
- Accuracy and precision: Some input devices, like mice and trackballs, are more accurate and precise than others.
- Speed: Some input devices, like keyboards, are faster than others.
- Ease of use: Some input devices, like keyboards and mice, are easier to use than others.
- Portability: Some input devices, like mice and keyboards, are portable.
