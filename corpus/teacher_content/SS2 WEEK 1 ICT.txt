INTRODUCTION
A modern computer system consists of a CPU, primary storage, secondary storage, input devices, output devices and communication devices.
THE STRUCTURE OF A COMPUTER SYSTEM
The parts making up the computer hardware can be classified into:
Input Devices: Input devices provide data and control signals to a computer. Examples include Keyboard, Mouse, Scanner, Microphone, Webcam, Joystick, and Touchscreen
Output Devices: Output devices translate data from the computer into a form humans can understand. Examples include Monitor, Printer, Speaker, Projector, Headphones
Storage Devices: Storage devices store data permanently or temporarily. Examples include Hard Disk Drive (HDD), CDs, DVDs, USB, Flash Drive, and Memory Card
Processing Devices: Processing devices interpret and execute instructions. The CPU, considered the brain of the computer, includes CPU, Motherboard, Sound Card, and Network Card.

THE CENTRAL PROCESSING UNIT (CPU)
The Central Processing Unit (CPU), often called the brain of the computer, is essential for interpreting and executing commands from hardware and software. It performs calculations, executes instructions, and manages data flow, directly impacting the computer's performance.
CPUs are found in a variety of devices, including desktops, laptops, tablets, smartphones, and flat-screen TVs. They are also known as processors or microprocessors. Modern CPUs are small and square, with short metallic connectors or pins on the underside. They attach to the motherboard via a CPU socket.
The clock speed of a CPU, measured in hertz (Hz), indicates how many instructions it can process per second. For instance, a 3.0 GHz CPU can process 3 billion instructions per second. CPUs can have multiple cores, such as single-core, dual-core, or quad-core, enhancing their processing capabilities.
FUNCTIONS OF THE CPU
The CPU executes instructions using a cycle known as the fetch-decode-execute cycle:
Fetch: The CPU retrieves an instruction from memory.
Decode: The control unit interprets the instruction.
Execute: The Arithmetic Logic Unit (ALU) performs the required operation, and the result is stored in a register or memory.
