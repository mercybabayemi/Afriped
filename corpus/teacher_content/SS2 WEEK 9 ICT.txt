COMPUTER DATA CONVERSION
Data Conversion in the conversion of computer dater from one format to another.
REGISTER
A register is a high-speed storage location within the CPU that holds data and addresses for processing. It may store a computer instruction, a storage address, or any data. The size of a register must match the instruction size of the computer, such as 32 bits for a 32-bit processor.
Address:
An address identifies a location in computer memory where data is stored. Each memory location, or storage box, has a unique number (address) used to access its contents.
Bus:
A bus is a communication pathway that allows multiple hardware components to exchange data. It reduces the need for separate communication lines by carrying all communications over a single channel.
Types of Bus
Address Bus: Carries memory addresses from the CPU to components like RAM and I/O devices, determining where data is read from or written to.
Control Bus: Sends control signals between the CPU and other components to manage data transfers, indicating whether data should be read or written and coordinating timing.
System Bus: A combination of the data, address, and control buses that transfers data, addresses, and control signals within the computer, connecting the CPU to main memory and internal components.

TYPES OF REGISTERS AND THEIR FUNCTIONS
Memory Data Register (MDR): - Holds the data that is being transferred to or from memory. It acts as a buffer for data being fetched from or written to memory.
Current Instruction Register (CIR): - Stores the instruction currently being executed or decoded by the CPU. It helps the control unit know which instruction to execute next.
Memory Address Register (MAR): - Holds the memory address of the data that the CPU needs to access. It specifies the location from which data is to be fetched or to which data will be written.
Program Counter (PC): - Keeps track of the address of the next instruction to be executed. It increments after each instruction fetch to point to the subsequent instruction.
Accumulator (ACC): - A special register used to store intermediate results of arithmetic and logic operations performed by the Arithmetic Logic Unit (ALU).

FACTORS AFFECTING SPEED OF DATA TRANSFER
Bus speed: The term "bus speed" refers to how quickly the system bus can move data from one computer component to the other. The faster the bus, the more data it can move within a given amount of time. It is measured in Hertz.
Bus width: The size of a bus, known as its width, is important because it determines how much data can be transmitted at one time. For example, a 16-bit bus can transmit 16 bits of data, whereas a 32-bit bus can transmit 32 bits of data
