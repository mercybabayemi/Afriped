COMPUTER GRAPHICS
Computer Graphics are graphics created using computers and more generally, the representation and manipulation of image/pictorial data by a computer.
Types of Graphic Packages
Painting Packages: These are used for creating and editing bitmap images (raster graphics). They are ideal for tasks like photo editing, digital painting, and image manipulation.
Drawing Packages: These focus on vector graphics, which use geometrical shapes like lines, curves, and polygons. They are perfect for logo design, typography, and illustrations.

COMPONENTS OF GRAPHIC PACKAGES
Drawing Window: The area where all drawing activities are performed.
Printable Page: Represents the layout and size of the printed output.
Menu Bar: Contains commands like File, Edit, View, and Help for various tasks.
Tool Bar: Icons providing quick access to related commands.
Standard Tool Bar: Common commands like New, Open, Save, and Copy.
Status Bar: Displays information about the selected object or current action.
Colour Palette: Used to select outline and fill colors for objects.
Ruler: Helps with accurate sizing and positioning of objects.
Title Bar: Displays document title and window controls (minimize, maximize, close).
BASIC TOOLS IN A GRAPHIC PACKAGE
1. Selection Tool: Select and manipulate objects.
2. Shape Tools: Create basic geometric shapes like rectangles, circles, polygons, and lines.
3. Freehand and Bezier Tools: Draw freeform lines and shapes.
4. Text Tool: Add and edit text in your design.
5. Fill Tool: Apply colors, gradients, and patterns to objects.
6. Outline Tool: Change the color, width, and style of an object's outline.
7. Shape Editing Tool (Node/Shape Tool): Edit the shape of objects by manipulating nodes and paths.
8. Zoom and Pan Tools: Navigate around your canvas.
9. Eyedropper Tool: Sample and apply colors from one object to another.
10. Eraser Tool: Remove parts of objects or images.
Examples:
Creating a Simple Logo:
Use the Rectangle Tool to draw the background.
Use the Ellipse Tool to create circular elements.
Use the Text Tool to add the brand name.
Adjust colors using the Fill Tool and Outline Tool.
Designing a Flyer:
Use the Rectangle Tool and Ellipse Tool to create shapes for images and text areas.
Use the Text Tool to add headings, subheadings, and body text.
Apply colors and gradients with the Fill Tool.
Arrange elements using the Selection Tool and align them with snapping and guidelines.
