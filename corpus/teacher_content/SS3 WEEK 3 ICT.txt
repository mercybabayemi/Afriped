INTRODUCTION TO WORLD WIDE WEB
World Wide Web Overview
The World Wide Web (WWW) is a system of interlinked hypertext documents accessed via the internet. It allows users to read and write information on web pages through devices connected to the internet.
Web development refers to the process of creating and maintaining websites. It involves coding, designing, and managing web applications to ensure they function properly online.
HISTORY OF THE WORLD WIDE WEB
1979 – 1991: Development phase led by Tim Berners-Lee at CERN. The web evolved from a project to enhance CERN communication into a global concept. 1989: Tim Berners-Lee proposed the web to facilitate file sharing at CERN.
1992 – 1995: Growth phase, marked by an increase in websites and users. 1995: Introduction of the internet in Nigeria through RINAF, marking the start of widespread internet use in the country.
1996 – 1998: Commercialization of the web led to the development of web technologies and widespread use.
1999 – 2001: The dot-com boom and burst due to overvaluation of tech stocks.
2002 – Present: The web becomes an integral part of everyday life and various industries.
BASIC TERMINOLOGIES
WWW: World Wide Web, the network of websites, resources, and users utilizing HTTP.
Website: A set of related web pages served from a single web domain.
Webpage: A document written in HTML, accessible through a browser.
Homepage: The main or initial page of a website.
Protocols: Rules governing communication between computers in a network (e.g., HTTP).
HTML: The primary language used for creating web pages.
Web Browser: Software that allows users to access and interact with web pages (e.g., Chrome, Firefox).

USES OF THE WORLD WIDE WEB
Advertising: Offers a cheaper, more flexible option compared to traditional media.
E-Learning: Facilitates learning through online platforms and courses.
Market Expansion: Helps businesses reach global audiences.
E-Commerce: Allows buying and selling of products online.
Convenience: Provides access to information and services from anywhere.
Growth and Revenue: Increases business sales and market reach.
