THE COMPUTER SYSTEM
A computer system can be defined as an electronic device that receives input, processes data according to stored instructions (software), and outputs results. It is capable of performing a wide variety of tasks, such as calculations, data processing, and control of external devices.
Components of a Computer System
A computer system is typically composed of the following key components:
Hardware
Software
Human ware
1. Hardware
Physical parts of the computer that can be touched and interacted. Examples include:
i. Input Devices: Such as keyboards, mice, and scanners, which allow users to enter data into the computer.
ii. Output Devices: Such as monitors, printers, and speakers, which display the results of processed data.
iii. Processing Devices: The Central Processing Unit (CPU) is the brain of the computer and handles data processing and execution of instructions.
iv. Storage Devices: Devices like Hard Disk Drives (HDDs), Solid State Drives (SSDs),  and USB flash drives that store data and programs either temporarily (RAM) or permanently.

SOFTWARE
Software refers to the instructions and programs that run on the computer’s hardware, telling it how to perform tasks. Examples include:
System Software: Such as operating systems (Windows, MAC OS, Linux) that manage hardware and software resources.
Application Software: Programs designed for users to accomplish specific tasks, like word processing or web browsing (Microsoft Word, Google Chrome).
HUMANWARE
Human ware refers to the users or people who interact with the computer system to input data, use software, and receive outputs. Without human interaction, the computer system cannot perform tasks in a meaningful way.
