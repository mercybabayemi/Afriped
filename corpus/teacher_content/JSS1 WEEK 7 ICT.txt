COMPUTER SYSTEM
A computer is an electronic device that accepts data, processes the data and supply the result in form of a meaningful information.
A computer performs three major functions which are:
It accepts data (input)
It processes data
It supplies data or information (output)
COMPONENTS  OF COMPUTER
Hardware
Software
Humanware
COMPUTER HARDWARE
The computer hardware is the part of a computer system that you touch, see and feel Computer Hardware is divided into two main categories namely:
System Unit: The system unit is the main part of a computer, typically a rectangular box, containing essential components like the motherboard, CPU, RAM, and drives.
Peripherals: Peripherals are external devices connected to a computer, such as input, output, storage, and communication devices like keyboards, monitors, printers, and modems.

COMPUTER SOFTWARE
Computer software is a collection of computer programs and related data that provide the instruction to a computer on what to do and how to do it.
Types of Software
Software is divided into two major classes:
System Software: - System Software is computer software designed to operate the computer hardware and to provide and maintain a platform running application software. It includes a combination of the following
Device drivers
Operating system like windows
Utilities
Application software: - Application software is computer software designed to help the user to perform a singular or multiple related specific tasks.
Examples of application Software are: Microsoft word, Microsoft Excel, Encarta Kids, Google Chrome, Candy Crush, etc.
HUMANWARE
Humanware refers to the individual using computer hardware and software. The user interacts with and gives directives to hardware using software to produce needed outputs. Examples include, Computer Engineer, System Analyst, Programmer, Software Engineer, etc.
CHARACTERISTICS OF COMPUTER
Computers operate at extremely high speeds, measured in nanoseconds and picoseconds.
Computers produce accurate results when given correct input.
Computers store large amounts of data in primary and secondary storage devices.
Computers can perform a wide range of tasks, from simple to complex.
Computers operate automatically with minimal human intervention.
